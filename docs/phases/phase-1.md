# Phase 1: Core Web Platform

## Objectives
Establish the foundational infrastructure and core application components of the Web-only ARMT-GAN Platform.

## Deliverables
- **Infrastructure Setup**: Initial Terraform scripts and Kubernetes cluster deployment.
- **Backend API**: Baseline FastAPI application with authentication and essential endpoints.
- **Web Frontend**: Next.js scaffolding with core routing and layout components.
- **Database Schema**: Initial PostgreSQL migrations via Alembic.
- **CI/CD**: Basic GitHub Actions for automated linting and testing.

## Status
In Progress.
